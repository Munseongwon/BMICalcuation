# BMICalcuation

### Promotion
- 이것은 현재 간단한 BMI 계산기로써 안드로이드 스튜디오를 사용하여 만들어보았습니다.
- 앱 아이콘 누를 시 4초동안 메인 엑티비티로 이동전 스플레시 엑티비티가 머물러 있도록 구현
- 메인 화면에는 성별, 키 설정, 몸무게, 나이를 UI로 구현
- 버튼을 이용하여 다음 엑티비티(BMI 측정 엑티비티)로 계산한 상태에서 넘어가도록 구현하였음
